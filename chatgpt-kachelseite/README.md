# Moodle Kachelseite

Eine einfache HTML-Seite mit fünf großen, quadratischen Kacheln.

## Dateien

- `index.html` – Seitenstruktur
- `style.css` – Gestaltung der Kacheln
- `script.js` – Klickanimation und Weiterleitung im aktuellen Tab
- `hintergrund.png` – Hintergrundbild der Kacheln

## GitHub Pages

1. Repository erstellen.
2. Alle Dateien in das Repository hochladen.
3. Unter **Settings > Pages** GitHub Pages aktivieren.
4. Als Quelle den Branch `main` und den Ordner `/root` auswählen.

Alle Kacheln führen zu:

`https://moodle.uni-siegen.de/mod/hvp/view.php?id=1492494&forceview=1`
