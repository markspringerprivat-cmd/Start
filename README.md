# Kursrahmen Gesprächsführung – v26

Statische HTML/CSS/JavaScript-Seite für GitHub Pages oder Einbettung in Moodle.

## Struktur

- `index.html`: vorgelagerte Kursübersicht mit den drei Blöcken Wahrnehmung, Dekomposition und Approximation
- `dekomposition.html`: ausgearbeitete Modulübersicht mit sieben Bausteinen
- `wahrnehmung.html`: Platzhalterseite
- `approximation.html`: Platzhalterseite
- `style.css`: Gestaltung, Responsivität, Hover-/Fokuszustände, Vorlesemodus
- `script.js`: lineare Kachelnavigation, dynamischer Inhalt, Vorlesefunktion, Moodle-Linkweiterleitung

## Änderungen in v26

- klarere Kursstruktur und Schrittanzeige auf der Startseite
- weicheres, moderneres Farbsystem
- aktive Themenkachel deutlicher markiert
- Breadcrumb auf der Dekompositionsseite
- Banner mit Untertitel und Baustein-Anzeige
- rechte Textbox als Lernziel-Liste
- Buttontext auf der Dekompositionsseite: „Modul starten“
- Hover- und Fokuszustände vereinheitlicht
- Escape-Taste bricht Vorlesen ab
- `#top`-Anker und automatisches Scrollen nach oben ergänzt

## Nutzung

Alle Dateien zusammen in ein GitHub-Repository hochladen. Für GitHub Pages muss `index.html` im veröffentlichten Ordner liegen.
