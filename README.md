# Dekomposition Layout

Statisches Webprojekt mit HTML, CSS und JavaScript.

## Dateien

- `index.html` – Grundstruktur der Seite
- `style.css` – Layout, Karten, Banner und Animationen
- `script.js` – Logik für das lineare Durchschalten der Themenkarten

## Nutzung mit GitHub Pages

1. Dateien in ein GitHub-Repository hochladen.
2. In GitHub unter `Settings` → `Pages` den passenden Branch auswählen.
3. Warten, bis GitHub Pages die Seite veröffentlicht hat.

Die CSS- und JavaScript-Dateien werden in `index.html` mit `?v=6` geladen, damit der Browser ältere Versionen nicht so leicht aus dem Cache nimmt.
