# Moodle Kachelseite

Diese kleine Webseite enthält fünf große, quadratische Kacheln am oberen Bildschirmrand.

## Dateien

- `index.html` – HTML-Struktur der Seite
- `style.css` – Gestaltung der Kacheln
- `script.js` – Klickanimation und Weiterleitung im aktuellen Tab
- `hintergrund.png` – vorhandene Bilddatei, aktuell nicht als Button-Hintergrund verwendet

## Linkziel

Alle Kacheln führen im aktuellen Browser-Tab zu:

https://moodle.uni-siegen.de/mod/hvp/view.php?id=1492494&forceview=1

## GitHub Pages

1. Dateien in ein GitHub-Repository hochladen.
2. In GitHub unter `Settings > Pages` GitHub Pages aktivieren.
3. Als Quelle den Branch `main` und den Root-Ordner auswählen.
